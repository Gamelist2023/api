AIチャットボットについて

このサイトでは、AIチャットボットがあなたの質問に答えてくれます。本来はGemini-Proが機能しますが、サーバーの問題でエラーが起きた時は予備のGeminiが使用されます本来はbingを使いたいのですがサーバの問題でgeminiにしました、また通常の会話なら3秒で回答されます(会話の内容によって変わります)。

新機能: 画像生成

!image 生成したい画像 例： !image 猫が踊る で画像を生成できます！画像生成に掛かる時間はおおよそ30秒です。また画像生成のエラーが起きることがありますが
何回やっても無理な場合一日の上限を超えてるかもしれません

サンプルコード

以下のコードはこのサイトを利用するためのサンプルコードです/リクエストを送るにはテキストは"https://webapi-8trs.onrender.com/ask?text=hello"

画像生成の場合は"https://webapi-8trs.onrender.com/generate_image?prompt=cat"という感じでリクエストしてください画像はbase64で返されます

Python

import requests url = "https://webapi-8trs.onrender.com/ask" params = {"text": "こんにちは", "user_id": "123"} response = requests.get(url, params=params) print(response.json())

JavaScript

const axios = require('axios'); const url = 'https://webapi-8trs.onrender.com/ask'; const params = { text: 'こんにちは', user_id: '123' }; axios.get(url, { params }) .then(response => { console.log(response.data); }) .catch(error => { console.error(error); });
