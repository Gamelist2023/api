AIチャットボットについて

このサイトでは、AIチャットボットがあなたの質問に答えてくれます。本来は自動でGPT-4が機能しますが、サーバーの問題でエラーが起きた時は予備のGPT-3.5が使用されるかもしれません。通常の会話なら3秒で回答されます。

新機能: 画像生成

!image 生成したい画像 で画像を生成できます！画像生成に掛かる時間はおおよそ30秒です。

更新:少しスタイル変えました

サンプルコード

以下のコードはこのサイトを利用するためのサンプルコードです：userIDを設定していない場合は自動でIPが使用されます

Python

            import requests

            url = "https://webapi-8trs.onrender.com/ask"
            params = {"text": "こんにちは", "user_id": "123"}

            response = requests.get(url, params=params)

            print(response.json())

JavaScript

            const axios = require('axios');

            const url = 'https://webapi-8trs.onrender.com/ask';
            const params = { text: 'こんにちは', user_id: '123' };

            axios.get(url, { params })
              .then(response => {
                console.log(response.data);
              })
              .catch(error => {
                console.error(error);
              });
